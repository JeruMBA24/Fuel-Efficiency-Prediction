Summary:

The Auto dataset from ISLP contains information about different cars and their performance details. It is mainly used to analyze and predict miles per gallon (MPG).

About Dataset:

The dataset includes details of around 400 cars.

It has 8 main numerical variables and 1 name variable.

The main target (response) variable is MPG (miles per gallon).

Other variables include:

Cylinders – Number of engine cylinders

Displacement – Engine size

Horsepower – Engine power

Weight – Car weight

Acceleration – Speed increase time

Model Year – Year of manufacture

Origin – Country of origin (USA, Europe, Japan)


Overall the dataset helps in understanding how factors like engine size, horsepower, and weight affect fuel efficiency (MPG).

Name – Car model name
